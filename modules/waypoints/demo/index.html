Waypoints

What?

Call provided functions when the specified elements transition from above the current window to above the current window, or from left to right.

Try scrolling past the red text or changing the offset

Why?

Make elements sticky, trigger class changes on different menu elements (e.g. in a sticky navbar) when key elements are passed

How?

<p ui-waypoints="callback">They see me scrollin...</p>

You can optionally pass an object to waypoints ui-waypoints. The following options are valid: which would override the detected y-offset of the element. Values can be either absolute 600 or offset from the calculated value -50 or +100.

AttributeDescription
enter Function that is called when transition from above to below or left to right. Passed the direction (down, right), and the element.
exit Function that is called when transitioning from below to above or right to left. Passed the direction (up, left), and the element.
both Function that is called when transitioning in either directionPassed the direction (left, right up, down), and the element.
offset Object which specifies the the detected y-offset of the element. Should be an object that looks like {"vertical": 10, "horizontal":15}. Values can be either absolute 600 or offset from the calculated value -50 or +100. In the latter case, they should be strings, and in the former, numbers.
